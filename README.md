# BUILD Equality Website

**Building Leadership and Employment Pathways for Women in Construction**

A British Council-funded partnership website between Northumbria University (UK), The Hashemite University, and Philadelphia University (Jordan).

## About

BUILD Equality is transforming the construction sector by empowering women to break barriers and advance into leadership and technical roles through workshops, training activities, and peer-learning opportunities.

## Features

- ✅ Fully responsive design (mobile, tablet, desktop)
- ✅ Modern, professional UI with smooth animations
- ✅ Detailed workshop information with embedded videos
- ✅ Team profiles and partner information
- ✅ Optimized for all screen sizes

## Technologies Used

- HTML5
- CSS3 (with responsive design)
- Bootstrap 5.3.2
- Google Fonts (Playfair Display, Inter)
- Bootstrap Icons

## Website Structure

- `index.html` - Main website file
- `logos/` - Partner organization logos
- Workshop images (flyers and speakers)
- Team member photos
- About section imagery
